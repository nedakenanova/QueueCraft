# QueueCraft

Queue management system built with ASP.NET Core Web API.

##  Technologies
- ASP.NET Core
- C#
- Layered Architecture
- Dependency Injection
- Swagger
- HTML + JavaScript (Fetch API)

## Features
- Create Service
- Get All Services
- Get Service By Id
- Create Ticket
- Get Tickets by Service
- Get Next Ticket
- Change Ticket Status

##  Architecture
- Domain layer
- Application layer
- Web API layer

##  Frontend
Simple HTML interface that communicates with the API using Fetch.
